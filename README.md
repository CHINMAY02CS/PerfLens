# 🚀 PerfLens

PerfLens is a frontend performance analyzer that helps developers understand how their websites perform and how to improve them.

It transforms raw performance data into **clear, actionable insights** — so you can identify bottlenecks and optimize faster.

## 🧠 What It Does

- Analyze any website using performance metrics  
- Visualize Core Web Vitals in a clean dashboard  
- Highlight performance bottlenecks  
- Provide actionable suggestions for optimization  

---

## 📊 Metrics Covered

PerfLens evaluates websites using key performance metrics:

- **LCP (Largest Contentful Paint)** – main content load time  
- **FCP (First Contentful Paint)** – first visible content  
- **CLS (Cumulative Layout Shift)** – visual stability  
- **TBT (Total Blocking Time)** – JS execution blocking  
- **TTFB (Time to First Byte)** – server response time

## ⚙️ Tech Stack

- **Frontend:** Next.js, React, TypeScript  
- **Styling:** Tailwind CSS  
- **Data Source:** PageSpeed Insights API  

---

## 🧠 How It Works

1. User enters a website URL  
2. PerfLens fetches performance data via PageSpeed API  
3. Data is transformed into structured metrics  
4. Custom logic analyzes bottlenecks  
5. Results are displayed with insights and suggestions
